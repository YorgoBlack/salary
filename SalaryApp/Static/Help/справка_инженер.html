Портал премий – web сайт, позволяющий сотрудникам просматривать начисленные и выплаченные по проектам премии.

Доступ к порталу премий возможен только из внутренней сети компании «Овен».

Просмотр портала осуществляется при помощи браузера, при переходе на адрес http://salary.kharkov.owen:9000/

 

ВНИМАНИЕ! По всем возникающим при работе с порталом вопросами, предложениями, замечаниями и ошибкам писать на почту kb_khar@owen.ua

В теме письма указать: «Портал работы с премиями. <Тип обращения>», где

<Тип обращения> - «Ошибка»/«Вопрос»/«Замечание» и т.д.

2. Основное меню портала

Поле 1: Просмотр начисленных и выплаченных премий с возможностью фильтрации по датам и проектам.

Поле 2: Вызов данной справки.

Поле 3: Просмотреть информацию учётной записи.

Поле 4: Выйти с портала.

 

 

3. Просмотр премий

Портал позволяет просмотреть начисленные премии в 2х представлениях:

– с возможностью выбора проекта и временного диапазона;

– сводный отчёт начисленных и выплаченных сумм по всем проектам.

Поле 5: Просмотреть премии с возможностью выбора проекта и временного диапазона (см. 3.1).

Поле 6: Просмотреть сводный отчёт начисленных и выплаченных сумм по всем проектам (см. 3.2).

 

 

3.1. Просмотр премий за выбранный период

Поле 7: Выбор периода, за который требуется просмотреть премии. Если даты не указаны – просматривается весь период.

Поле 8: Выбор проекта, по которому требуется просмотреть премии.

Поле 9: Применить фильтр выбора полей 7 и 8.

Поле 10: Суммарные данные. Зависят от выбора в полях 5 и 6.

                               «Остаток к выплате, грн» - часть премии, не выплаченная на данный момент.

«Всего выплачено, грн» - сумма всех уже произведенных выплат по выбранному проекту.

Поле 11: Отображение данных, в соответствии с выбором в полях 5 и 6.

 

 

 

3.2. Просмотр премий по проектам

Поле 12: Список всех проектов, по которым начислялись премии.

Поле 13: Сумма премии, начисленная по проекту.

Поле 14: Сумма выплаченной премии на данный момент.

Поле 15: Сумма невыплаченной премии на данный момент.

Поле 16: Переход к подробному (по месяцам) просмотру выплат премии по проекту.
