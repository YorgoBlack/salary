Портал премий – web сайт, позволяющий сотрудникам просматривать начисленные и выплаченные по проектам премии.

Доступ к порталу премий возможен только из внутренней сети компании «Овен».

Просмотр портала осуществляется при помощи браузера, при переходе на адрес http://salary.kharkov.owen:9000/

 

ВНИМАНИЕ! По всем возникающим при работе с порталом вопросами, предложениями, замечаниями и ошибкам писать на почту kb_khar@owen.ua

В теме письма указать: «Портал работы с премиями. <Тип обращения>», где

<Тип обращения> - «Ошибка»/«Вопрос»/«Замечание» и т.д.

 

Права, доступные Вам на портале:

- просмотр начисленных Вам премий (как уже выплаченных, так и планирующихся к выплате);

- формирование заявок, для начисления премий любому сотруднику КБ по проектам Вашего АН;

- просмотр всех заявок на начисление премий Вашего АН (как поданных, так и выполненных);

- просмотр начисленных премий по всем проектам Вашего АН;

- просмотр премий всех сотрудников Вашего АН.

3. Просмотр собственных премий

Портал позволяет просмотреть начисленные премии в 2х представлениях:

– с возможностью выбора проекта и временного диапазона;

– сводный отчёт начисленных и выплаченных сумм по всем проектам.

Поле 3.1: Просмотреть премии с возможностью выбора проекта и временного диапазона (см. Пункт 3.1).

Поле 3.2: Просмотреть сводный отчёт начисленных и выплаченных сумм по всем проектам (см. Пункт 3.2).

 

3.1. Просмотр премий за выбранный период

Поле 3.1.1: Выбор периода, за который требуется просмотреть премии. Если даты не указаны – просматривается весь период.

Поле 3.1.2: Выбор проекта, по которому требуется просмотреть премии.

Поле 3.1.3: Применить фильтр выбора полей 3.1.1 и 3.1.2.

Поле 3.1.4: Суммарные данные. Зависят от выбора в полях 3.1.1 и 3.1.2.

                                «Остаток к выплате, грн» - часть премии, не выплаченная на данный момент.

«Всего выплачено, грн» - сумма всех уже произведенных выплат по выбранному проекту.

Поле 3.1.5: Отображение данных, в соответствии с выбором в полях 3.1.1 и 3.1.2.

 

3.2. Просмотр премий по проектам

Поле 3.2.1: Список всех проектов, по которым начислялись премии.

Поле 3.2.2: Сумма премии, начисленная по проекту.

Поле 3.2.3: Сумма выплаченной премии на данный момент.

Поле 3.2.4: Сумма невыплаченной премии на данный момент.

Поле 3.2.5: Переход к подробному (по месяцам) просмотру выплат премии по проекту.

Поле 4.9: Информация о заявке – изменить текущее состояние (статус) отдельной заявки.

4.1. Алгоритм начисления премии

1)    Создать заявку на премию. Заявка получает состояние «Не отправлена». В данном состоянии заявка может быть отредактирована.

2)   Отправить созданные заявки на согласование Директору.

3)   Директор, рассмотрев поданные заявки, имеет возможность:

      - Согласовать заявку. Заявка получает состояние «Утверждена» и попадает на обработку Администратору портала.

      - Отклонить заявку. Заявка получает состояние «Отклонена», будет доступна к просмотру в списке заявок, но больше не может быть кем-либо обработана.

      - Отправить заявку на доработку. Заявка получает состояние «На доработке» и может быть изменена постановщиком (после изменения параметров заявки, она снова должна быть переведена в состояние «На утверждении»)

 

4.2. Заявка на начисление премии

Для начисления премий на портале реализован механизм «Заявок». Создать новую заявку и просмотреть предыдущие возможно на странице «Начисление премий» -> «Создать заявку».

Заявка может находиться в одном из следующих статусов (состояний):

 

4.2.1. Создание новой заявки

Поле 4.2.1.1: Указать название проекта, по которому начисляется премия. Доступна всплывающая подсказка по ранее созданным проектам своего АН.

Поле 4.2.1.2: Указать сумму начисляемой человеку премии. Указывать сумму премии можно как в часах, так и в гривнах.

Поле 4.2.1.3: Указать адресата начисляемой премии. Доступна всплывающая подсказка.

Поле 4.2.1.4: Указать наличие согласования начисляемой человеку премии с ПМ.

Поле 4.2.1.5: В случае необходимости, прикрепись к заявке файл (например, файл переписки с ПМ, в котором присутствует согласование премии).

Поле 4.2.1.6: В случае необходимости, внести текстовый комментарий.

Поле 4.2.1.7: При установке данной «галочки» заявка, при создании, получит статус «На утверждении» и будет автоматически отправлена Директору.

Поле 4.2.1.8: Создать новую заявку

Поле 4.2.1.9: Отказаться от создания новой заявки и вернуться к предыдущему окну.

 

 

5. Просмотр выплат АН

Портал позволяет просмотреть выплаченные в рамках АН премии в 2х представлениях:

- по всем проектам своего АН;

- по сотрудникам своего АН.

Поле 5.1: Просмотр премий, выплаченных сотрудникам.

Поле 5.2: Просмотр премий, выплаченных по проектам.

 

5.1. Просмотр выплат по проектам своего АН

Поле 5.1.1: Список всех проектов АН, по которым производилось начисление премий.

Поле 5.1.2: Поле зарезервировано для дальнейших разработок.

Поле 5.1.3: Общая сумма премий, начисленная по данному проекту (всем сотрудникам).

Поле 5.1.4: Общая сумма премий, выплаченная по данному проекту (всем сотрудникам).

Поле 5.1.5: Общая сумма премий, не выплаченная по данному проекту (всем сотрудникам).

 

5.2. Просмотр выплат по сотрудникам своего АН

Поле 5.2.1: Список сотрудников АН.

Поле 5.2.2: Название АН.

Поле 5.2.3: Проект, по которому начислялась премия сотруднику.

Поле 5.2.4: Сумма выплаченной сотруднику премии по данному проекту.

Поле 5.2.5: Сумма премии, которая будет выплачена сотруднику по данному проекту.

Поле 5.2.6: Сумма выплаченных премий с разделением по месяцам.
